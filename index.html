RFLCT - Smart Mirror

RFLCT is a smart mirror utilizing the Amazon Alexa API, voice recognition and face detection. RFLCT is voice activated and will provide current news from websites such as USA Today, Reddit, and CNN. Our team developed the application and built a mirror within a span of 9 days.

View Demo

Tech Stack: JavaScript, Node, Electron, Snowboy API, Kairos API, Amazon Alexa API + Lambda, Python, Raspberry Pi

HCO Track

HCO Track is an in-house application that helps track Home Care Organization applications and assist them in abiding by the Home Care Services Consumer Protection Act. I engineered the UI/UX to allow colleagues to efficiently track HCO cases and query specific statistics. Using SQL and Visual Basic, this application maximized productivity and efficient querying for thousands of important applications.

Tech Stack: SQL, Visual Basic, Microsoft Access

dotChomp

dotChomp is an endless runner mini-game built for Web browsers. Utilizing Phaser and vanilla Javascript, this application tracks all scores where the top 30 will be projected on the main page.

Play Now

Tech Stack: JavaScript, Ruby, Sinatra, Heroku, Phaser.io

HaystackTV Roku


Haystack TV's Roku App curates news related videos for users across the world. Haystack learns about the user's interests in order to surface the news that matters to them. Developed in XML and BrightScript, I built most of the UI and functionality to deliver content effectively. While still in production, this app is projected to immediately reach over 20,000 users when fully complete.

View Roku Channel

Tech Stack: XML, BrightScript, Eclipse, Roku

FastBall

An Android game that utilizes mobile accelerometers to tilt the ball in a direction to either dodge obstacles or obtain items. This project is in early development.

Tech Stack: C#, Unity 5, Android SDK, Xamarin Studio

PoE Currency Exchange

PoE Currency Exchange is a web application that tracks and converts in-game currency from an online RPG with over 1.5 million total users. This application stack utilizes React, Redis, Amazon Web Services, and Bootstrap to display averages of 200,000+ unique records a month.

View Website

Tech Stack: Ruby On Rails, React, Redis, Amazon EC2 + RDS, Bootstrap, Passenger + Nginx

LabTrakk

LabTrakk is a Ruby on Rails application that permits users to propose lab experiments utilizing the scientific method. Users can vote and leave comments on their favorite lab experiments. The top proposals will be at the top of the list where others can test the experiment.

View Website

Tech Stack: Ruby, JavaScript, Sinatra, Heroku
